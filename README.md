## Getting Started as Contributor  

To get a local copy up and running follow these simple steps.

### Installation

 - Clone the repo
 - `https://github.com/ragify/Alphahacks-project`
 - Install the necessary packages
 - `pip install -r requirements.txt`
 - Run the application
 - `python main.py`
